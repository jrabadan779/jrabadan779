### Proyectos destacados

**[Control de Estabilidad PID para Quadcopter](https://github.com/jrabadan779/control-estabilidad-pid-dron)** — MATLAB / Simulink

Modelado dinámico y sintonización de un controlador de actitud. Sobreimpulso reducido un 92 % y tiempo de establecimiento de 0,52 s, verificado por doble método (integración numérica y Control System Toolbox).

**[Pulsioxímetro SpO2 Embebido](https://github.com/jrabadan779/pulsioximetro-spo2-embebido)** — C bare-metal (ATmega328)

Firmware no bloqueante (I2C, Timers, ISR) y algoritmo ratio-of-ratios en punto fijo. Reducción de SRAM del 49 % frente a coma flotante (medido con avr-gcc), validado contra el dataset clínico PhysioNet (MAE 2,1 %).

**[Mitigación de Interferencias ISM 2.4 GHz](https://github.com/jrabadan779/sdr-ism-noise-mitigation)** — SDR / GNU Radio

Caracterización y mitigación de ruido en la banda de 2.4 GHz con filtrado FIR digital. Mejora de relación señal-ruido de 12,44 dB, medida con metodología señal-ON/señal-OFF.
